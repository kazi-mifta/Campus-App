## Features
  - Instant Notifications
  - Dynamic Calendar
  - Dedicated Website For News
  - Geeks Section For Help
  - Dedicated Cloud Space
  - Photography Website
  - Map Of University


## Things I have used to create this
 - Adobe Photoshop
 - Axure RP
 - Google Firebase
 - Android Studio
